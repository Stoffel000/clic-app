Description

This application is part of a research initiative to enable fully autonomous closed-loop glucose control of immunocompromised patients.

Features

  • Searchable database of the International Table of Glycemic Index and Glycemic Load Values.
  • Graphically displays post-prandial prediction of glucose kinetics for normal, T2DM and T1DM patients.
  • Ability to create and save custom meals for reuse later on.
  • Multi-meal predictions
  • Ability to factor in physical activity.
  • Recommendations for maintaining normal glucose levels if meals exceed or undershoot target bounds.(_Not for primetime)_

In the Works

  • Image detection and classification of meals.
  • Feature extraction from barcodes.
  • Real-time glucose level tracking.
